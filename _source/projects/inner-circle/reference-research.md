## 5 条核心结论（TL;DR）

1. **全球同源** — Shark Tank / Dragon's Den / 合伙中国人 / 投吧 全都源自日本《Money Tigers》。→《入局》可正当打「全球模式的大马华人篇」。
2. **「国际级」的第一杠杆 = 钱是真的** — 国际版都没有冠名，靠"真钱真股权真风险"建立可信度。logo 一上谈判桌就掉价。
3. **中国经验** — 走"博弈紧张 + 一点殿堂分量"；**选人重镜头感/金句力**（《我是创始人》就是选人闷而崩）；用真老板当评委；主动埋金句时刻。
4. **本地洞察** — 华语 Shark-Tank 赛制型**几乎是蓝海**（本地只投吧一档）；**银行冠名是已验证模型**（Affin/Alliance）；本地没人掏真股权 → **《入局》做真股权 = 差异化**。
5. **3 个待《入局》拍板** — ①投资是真钱还是象征 ②赛制"搞定1位"vs"全或无" ③气质偏博弈紧张(B)还是殿堂权威(A)。

> 深度拆解见项目文件：`入局 — 参考节目拆解 (中国+国际).md` · `入局 — 本地节目拆解 (本地).md`

## 节目库

### 🌍 国际标杆

**Shark Tank** `国际·赛制`
- **平台/年份**：美国 ABC · 2009–
- **一句话**：投资路演的金标准，5 位"鲨鱼"当场用自己的钱买股权
- **Framework**：无主持只旁白（评委即权威）· 5 鲨鱼轮换 · 走廊 + 一排王座
- **Flow**：走廊入场 → "求 $X 换 Y%" → 质询估值 → 报价 → "I'm out"；搞定 1 位即成交
- **Positioning**：周五全美第一 · 5 座艾美奖
- **Content**：消费品为主 · 反差人设 · "the one that got away"
- **Sponsor**：无冠名 · 真钱真股权 = 内容本身 · 全球模式授权
- **⭐ 对《入局》**：标准化「ASK」字幕包装 · 真股权 = 可信度 ·（注：入局已定有主持 Jeff，故走"权威主持 + 大佬"而非无主持）
- **Researcher**：Zheng · 2026-07 | **Source**：Wikipedia

**Dragon's Den** `国际·赛制`
- **平台/年份**：英国 BBC / 加拿大 CBC · 2005–
- **一句话**：同源更冷峻版，桌上堆真钞，"全或无"
- **Framework**：5 条"龙" · 记者旁白 · 桌上钞票堆道具 · 拾级进"den"
- **Flow**："求 £X 换 Y%" → 盘问 → **全或无**（拿满全额否则空手走）
- **Positioning**：升上 BBC One = 血统印章 · 记者主持显严肃
- **Content**：更多实体小企业 · 冷静 boardroom 调 · "vanishing deals"风险
- **Sponsor**：BBC 执照费 = 零广告零赞助（最高级形态）
- **⭐ 对《入局》**：钞票堆道具设计大马版 · "全或无"更紧张 · 升频道 = 血统叙事
- **Researcher**：Zheng · 2026-07 | **Source**：Wikipedia

### 🇨🇳 中国

**合伙中国人** `中国·赛制`
- **平台/年份**：深圳卫视 · 2016 / 2017
- **一句话**：中国版 Shark Tank，千亿身价大佬团当场抢投
- **Framework**：6 人投资人轮换池 · 无主持 · 正版引进
- **Flow**：「十分钟融资」路演 1–3min → 质询 → 抢投 / 集体拒绝两极 → 议价
- **Positioning**："创投版《美国偶像》"· 四段式 slogan 圈四类人
- **Content**：真创业者生活化 · 争议人物情感张力 · 大佬花絮做切片
- **Sponsor**：冠名点融网 / 雅堂小超（金主 = 创投品牌同频）· 真金白银
- **⭐ 对《入局》**：投资人天团强人设 · 核心机制起响名 · 实录狠剪只留两极
- **Researcher**：Zheng · 2026-07 | **Source**：百度百科 / 界面

**赢在中国** `中国·赛制·殿堂鼻祖`
- **平台/年份**：CCTV / 江苏卫视 · 2006–
- **一句话**：商战真人秀鼻祖，马云、柳传志当评委
- **Framework**：主持王利芬 · **实业大佬 × 财务投资人两派对撞** · 108→36→12→5 晋级
- **Flow**：BP 阐述 → 答辩 → 点评 → 投票；2013 版升级 24h 商战任务
- **Positioning**：slogan「励志照亮人生，创业改变命运」· 主题曲大佬联名作词
- **Content**：已成名企业家 · 校友叙事长尾招牌 · 金句 = 病毒资产
- **Sponsor**：中国联通冠名 · 奖励 = 1000 万公司 CEO + 20–50% 股权
- **⭐ 对《入局》**：评委两派对撞 · 锁魂 slogan · 奖励 = 真投资 · 经营校友叙事
- **Researcher**：Zheng · 2026-07 | **Source**：百度百科 / 虎嗅

**创业英雄汇** `中国·赛制·央视克制`
- **平台/年份**：CCTV 财经 · 2014–
- **一句话**：央视"创业实战公开课"，克制权威
- **Framework**：主持陈伟鸿 = 权威锚点 · ~30 职业投资人天团亮灯
- **Flow**：5min 路演 → 亮灯 → 「100 秒快问快答」→ 意向融资额打屏
- **Positioning**："公开课"双创国家叙事 · 累计约 31 亿意向额
- **Content**：硬科技 / 实体 · 全国海选 + 高校巡回 · 专业质疑非撕扯
- **Sponsor**：邮储银行联办 · 平台型变现
- **⭐ 对《入局》**：权威主持定调（呼应 Jeff）·「100 秒问答 + 亮灯」直接抄 · 银行冠名
- **Researcher**：Zheng · 2026-07 | **Source**：央视 / 百度百科

**天生我有才** `中国·赛制·明星跨界`
- **平台/年份**：浙江卫视 · 2015
- **一句话**：首档明星跨界创投真人秀
- **Framework**：明星合伙人（胡海泉 = 真天使投资人）· 背后一线 VC 机构
- **Flow**：明星当伯乐挑 2 项目 → 下场体验 → 携团队面 5 大机构终极考核
- **Positioning**：明星流量 + 情感代入 · 踢馆 / 连胜机制
- **Content**：名人分工 = 伯乐 + 体验 + 竞争；真金白银由机构裁决
- **Sponsor**：京东既是投资团又是冠名 = 以投资人身份植入（最高级）
- **⭐ 对《入局》**：若放名人照此分工（名人管流量、机构管钱）· 冠名商 = 场上资源方
- **Researcher**：Zheng · 2026-07 | **Source**：百度百科 / 知乎

**我是创始人** `中国·赛制·⚠️反面案例`
- **平台/年份**：浙江卫视 · 2016
- **一句话**：真 CEO 亲自下场竞技，叫好不叫座（豆瓣 7.8 但收视惨败）
- **Framework**：董明珠 / 任泉 / 王小川等下场 · 无主持 · 收视对赌
- **Flow**：保密任务（海底捞当服务员 / 街头卖货）→ 复盘 → 淘汰
- **Positioning**：⚠️ 放传统黄金档错配 → 教训 = **社媒优先做 KPI**
- **Content**：反差去光环 · breakout IP 孙涛勇→微盟 · ⚠️ 选人重项目轻镜头 → 崩
- **Sponsor**：⚠️ 无强冠名 = 纯平台补贴（败因之一）
- **⭐ 对《入局》**：选人镜头 / 金句力优先 · 每集倒推切片名场面 · 别只靠平台补贴
- **Researcher**：Zheng · 2026-07 | **Source**：界面 / 豆瓣

### 🇲🇾 马来西亚本地

**投吧！合伙人 (The Sandbox)** `本地·华语·赛制·核心`
- **平台/年份**：Astro AEC × MyStartr · 2023 · 两季
- **一句话**：全马首档中文众筹真人秀，几乎是《入局》的本地前身
- **Framework**：主持 S1 小马 / S2 颜江瀚 · 3 大佬导师（Cuckoo / Tealive / Khind→CARSOME）· 周六黄金档
- **Flow**：电梯路演 → 商业模式九宫格 → 众筹团队赛 → 决赛向 100 位公众投资人；S2 融资额游戏化
- **Positioning**：全马首档中文众筹真人秀 · 绑定股权众筹平台
- **Content**：项目超广接地气 · 用真老板当导师建可信度
- **Sponsor**：冠名 Affin Bank · 与 MyStartr 变现绑定 · 旗舰战绩众筹近 RM1000 万
- **⭐ 对《入局》**：真老板 + 银行冠名 + 黄金档 = 已验证 · 绑定众筹后端 · 融资额游戏化
- **Researcher**：Zheng · 2026-07 | **Source**：Astro 官方 / Vulcan Post

**Biz Smart Challenge** `本地·英语·银行赛制`
- **平台/年份**：Alliance Bank × EcoWorld · 2018–
- **一句话**：大马首个由金融机构做的商业真人秀
- **Framework**：主持 Freda Liu · 导师 + 评审分层 · Astro Awani + AXN 双平台
- **Flow**：报名 → 20 强 → 商业培训 → 12 强 → 系列挑战 → 每周淘汰 → 直播决赛；「60 秒电梯演讲」
- **Positioning**：首个金融机构真人秀 · 明打 drama / 淘汰当卖点
- **Content**：成长期 SME 消费品牌 · 校友榜 = 最强资产（Christy Ng / BloomThis / Amazin' Graze）
- **Sponsor**：奖励 = 现金 + 银行融资额度（非股权）· 近年转 iQIYI 纪录片
- **⭐ 对《入局》**：三层角色分工 · 60 秒高压环节 · 校友追踪机制 ·（做真股权 = 对它的差异化）
- **Researcher**：Zheng · 2026-07 | **Source**：says.com / Vulcan Post
